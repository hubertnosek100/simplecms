Tags

Simpletext

Simpletext tag is for editable areas, you want to change in a fly.
It is watched and saved after 1 sec.

<simpletext> </simpletext>

You should add uuid attribute and than this tag will be remembered.

<simpletext uuid="example-uuid"> </simpletext>

All of components with uuid are remembered in database.
You can see that in database dashboard.

Every time you make new uuid to tag element you're creating new entity.
There is no automatic tool to remove unused tags with dead uuid.

Simpleimage

Simpleimage tag you may also change in the fly.
Click to change url.

<simpleimage> </simpleimage>

Remember to add uuid attribute

<simpleimage uuid="example-uuid"> </simpleimage>


You can observe that, in a <simpleimage> is added node with tag <img>
and with src attribute you had entered.
The class attribute is copied in it.

<simpleimage uuid="example-uuid" class="simple">
  <img class="simple"></img>
</simpleimage>

Simplevideo

Simplevideo is like simpleimage.
Click to change url.

<simplevideo> </simplevideo>

Remember to add uuid attribute

<simplevideo uuid="example-uuid"> </simplevideo>

Here is the same situation like in <simpleimage> tag.
The class attribute is copied in <simplevideo> tag.

<simplevideo uuid="example-uuid" class="simple">
  <video class="simple"></video>
</simplevideo>
